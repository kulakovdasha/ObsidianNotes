## Purpose
Generate Instagram Slide Post via HTML.

## Rules
- Canvas: **1080×1350px** portrait
- Slides: **3–6** total, first slide is always the Cover
- Content source: **"New Slides"** folder
- Don't add elements without asking first
- No multi-color words
- Each slide must have a different layout — same visual language, different composition

## Visual System

### Typography
| Element | Font | Size | Weight | Color | Letter-spacing | Line-height |
|---|---|---|---|---|---|---|
| Cover title | Rubik Black | 96px | 900 | #FFFFFF | -1px | 1.05 |
| Slide title | Rubik Black | 76px | 900 | #FFFFFF | -1px | 1.05 |
| Cover subtitle | Inter | 38px | 400 | rgba(255,255,255,0.6) | 0 | 1.47 |
| Slide text | Inter | 44px | 400 | rgba(255,255,255,0.6) | 0 | 1.47 |

No ALL CAPS.

### Colors
- Background: `#000000`
- Primary text: `#FFFFFF`
- Secondary text: `rgba(255,255,255,0.6)`
- Footnotes: `rgba(255,255,255,0.35)`
- Borders: `rgba(255,255,255,0.12)`

### Spacing
- Padding: `80px` top · `72px` sides · `100px` bottom
- Slide counter: top-right, `52px / 72px` from edges
- Watermark `raccoonpro.realestate`: bottom-right, `40px / 72px` from edges

### Cards
Used when content is a list or comparison. Border `1.5px solid rgba(255,255,255,0.12)`, radius `16px`, padding `36px 48px`, gap `24px`.
